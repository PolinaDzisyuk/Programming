## Отчёт по лабораторной работе Отладчик
1.
    Указатель инициализирован NULL, в функцию передаётся его копия, и память выделяется только под копию.\
    ![alt text](</lab_11/screenshots/Снимок экрана 2025-12-19 101318.png>)
    При вызове функции init(arr, n); - arr равен NULL.
    В функцию init передаю указатель на int *arr разыменовываю его и выделяю память под указатель, а не под его копию.
2.
    Происходит перполнение массива str. У нас 4 числа плюс '\0' = 5 символов. Нуль-терминатор записывается в num и портит значение.
    Увеличиваю массив, чтобы хватило памяти под нуль-терминатор.
3.
    Не хватает скобок. Без них макрос раскрывается как z = y + 1 * y + 1.
    ![alt text](</lab_11/screenshots/image.png>)
    Добавила скобки.
4.
    Каждая пара элементов обменивается дважды, как итог, матрица не транспонируется.
    ![alt text](</lab_11/screenshots/image-1.png>)
    В функции transpose меняю int j = 0 на int j = i + 1, чтобы поменять только элементы выше главной диагонали с элементами ниже главной диагонали.
5.
    XOR Cipher
    В цикле использовалось условие i <= len, что приводило к чтению лишнего байта.
    Условие было исправлено на i < len.
    Caesar Cipher
    Условие text[i] > 'a' && text[i] < 'z' пропускало буквы 'a' и 'z'. Также при отрицательном сдвиге результат % 26 может быть отрицательным.
    Исправлены границы на >= и добавлен алгоритм нормализации сдвига: (text[i] - 'A' + shift) % 26 + 'A'.
    Vigenere Cipher
    Формула text[i] + shift приводила к выходу за пределы таблицы ASCII.
    Был добавлен учёт регистра и циклическая арифметика % 26.
    RC4-like Cipher
    В цикле инициализации массива S стояло условие i < 256 + 32, что вызывало переполнение буфера.Граница строго ограничена 256.
    Simple Hash
    sizeof(str) для указателя всегда возвращает 4 или 8 байт, а не длину строки.
    Для получения длины строки использована функция strlen(str).
    Key Generation
    srand(time(NULL)) вызывался внутри функции. При быстром вызове функции дважды время не менялось, и генерировались идентичные ключи.
    srand вынесен в main(), чтобы инициализация происходила один раз.
